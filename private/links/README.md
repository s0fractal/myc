# Local Links

This directory is reserved for local symlinks into tool and storage roots.

Potential links:

```text
codex  -> ~/.codex
gemini -> ~/.gemini
drive  -> Google Drive mount
```

Links should be created intentionally by the local owner. Protocol drafts should
refer to descriptor hashes and capability records, not these local paths.
